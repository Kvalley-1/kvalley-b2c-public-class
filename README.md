## 二、Review Personas（客戶視角審查 agent）

兩支在 Claude Code 上跑的 persona agent，用客戶眼睛審查智谷對外文案 / 課綱 / EDM / LP。檔案位於 `review-personas/`：

| Persona | 視角 | 適用 |
|---|---|---|
| **陳總** | 中小企業老闆 / 副總 / 特助（決策者） | 給老闆 / 商會場 / 副總看的文案 |
| **林主管** | 跨職能中階部門主管（業務 / 行銷 / 訓練 / 營運 / IT / 工程） | 對中階主管的通用文案 / LP / EDM |

每支由兩個檔組成：
- `SKILL.md`：Claude Code 觸發描述（何時叫這支 persona）
- `CLAUDE.md`：persona 完整定義（人設、四關評分、客戶語料來源；樣本人物已脫敏）

要在自己的 Claude Code 用：
1. `SKILL.md` 放進 `.claude/skills/<name>/`
2. `CLAUDE.md` 放進對應 agent 資料夾（例如 `agents/陳總/`）
