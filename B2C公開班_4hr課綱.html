主帶：Jimmy + Dennis

歡迎與課程地圖

  • 雙講師自介（精簡）
  • 今天 4 小時流程說明
  • 報名時選的實作情境分組公告
50 min Section 1
主帶：Jimmy

趨勢與框架：老闆要佔住哪一層

  • 寒武紀比喻破冰
  • 八套工具行走天下（GPT / Gemini / Sheets / NotebookLM / Make / Notion / Flow / Claude Code）
  • 三套工具搞定一件生意大事（5 種真實組合）
  • 人機協作 L1–L7：你公司現在卡在哪一層
  • 用定製西裝心態思考個性化工作流
  • AI 為何不穩定
  • Token = AI 經濟計量單位（有刷卡 vs 沒刷卡的差別）
  • 標工 → 流量 → 算力：經濟典範轉移
45 min Section 2
主帶：Jimmy 主 / Dennis 知識點

Claude Code 卡牌：排出你的工作流地圖

  • 每桌一組卡牌：情境卡 + 工具卡（MCP / Skill）
  • 玩法：抽情境 → 挑工具 → 排順序 → 同桌互看誰排得最像
  • 講師桌邊巡視點評：哪些組合會卡、哪些會跑通
10 min 中場

休息、加水、檢查筆電環境

50 min Section 3
主帶：Dennis 主 / Jimmy 旁註

Claude Code 基礎 + MCP/Skill

  • CLAUDE.md 是什麼：給 AI 看的組織規則 / 員工手冊——你已經懂帶員工、寫員工手冊，這是同一件事的延伸
  • 組織設計規則怎麼養：身份脈絡 / 工作邊界 / 對話原則 / 出錯怎麼處理（含 Claude Code 基本設定）
  • MCP 用法：怎麼接外部資料 / 系統
  • Skill 用法：什麼時候用、怎麼用
  • Dennis 現場示範跑一條完整流程
60 min Section 4
主帶：Dennis 老師群分桌 / Jimmy 巡桌

實作三選一：建立你的幕僚團隊

  • 5 min：Dennis 統一講實作 SOP（開範本 → 替換你的資料 → 跑指令 → 看回應 → 修指令）
  • 50 min：學員依範本實作——拿課前下載的「CLAUDE.md 老闆幕僚通用版基底 + 情境模組」改成你公司版本
  • 報名時三選一：事業經營夥伴 / 外包業務團隊 / 行銷經理
  • Dennis 老師群（你的 1-on-1 講師）依產業 + 情境分桌指導，每組 5 學員 / 2 位專屬講師
  • 卡關時老師到桌邊 1 對 1 解
  • 5 min：講師標 demo 候選人，為收尾段過渡
15 min 收尾
主帶：共同

demo 分享 + 1-on-1 銜接

  • 2–3 位老闆 demo 分享
  • 1-on-1 預約方式、流程說明（講師全程在場，不另做自介）
  • 課後問卷、收 LINE 群、發資源包
