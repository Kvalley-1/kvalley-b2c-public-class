🎯 任務卡 — 老闆的商業挑戰（10 張，verbatim 校準）

1. 新一季產能計畫
「我想建一個新的生產計畫，幫我規劃產能 + 按往常規矩評估 + 生成報告 + 通知各部門主管幾點前回報」

策略產品財務特助

產能 brief + 通知 Email 模板

2. 三案同時要趕
「手上 3 個案子要趕，盤一下哪個先動、哪個派誰、哪個能砍」

"I'm spread very thin"（Indie Hackers founder Pete）
預期動員
策略特助
你帶走
3 案優先順序矩陣 + 派員建議
🎩
3. 派 team 分頭做
「這週要同時做行銷 + 業務 + 財務 + 提案，派 team 分頭做給我整合版」

"Wear all the hats, then learn how to take them off"（Channing Allen）
預期動員
行銷銷售財務特助
ChatGPT 為什麼做不到
沒法真平行派多個 agent

📝
4. 合興提案不能 workslop
「合興提案，根據他們真實痛點 + 我們過去類似案口氣 + 不能有 AI workslop 感」

銷售策略特助
你帶走

5. Q3 業績不如預期
