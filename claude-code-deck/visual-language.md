## 1. 卡牌規格

| 項目 | 規格 |
|------|------|
| 方向 | 直式（Portrait） |
| 尺寸 | 標準撲克牌 63×88mm **放大 20%** → **76×106mm** |
| 圓角 | 建議 4-6mm（視印刷廠規格調整）|
| 材質建議 | 300gsm 以上銅板 + 霧膜（防指紋、質感佳）|

---

## 2. 整體視覺風格

**溫暖商務風（Warm Business）**

- 米色 / 象牙底（非純白）
- 結構化欄位排版
- 定位：高階商業工具，不是遊戲牌，不是工程師教科書
- 對標參考：LEGO Serious Play / IDEO Method Cards / Business Model Canvas

---

## 3. 正面（Front）設計規則

### 版面結構（由上到下）

```
┌─────────────────────────┐
│                         │
│  卡型圖示 + 卡名          │  ← 頂端：最重要資訊，米色底，無色塊
│  （最大字，清晰）          │
│                         │
├─────────────────────────┤
│                         │
│  內容欄位（結構化）        │  ← 主體：各卡型對應的資訊欄位
│                         │
│                         │
├─────────────────────────┤
│ ██ [卡型名] [編號] [K] ██│  ← 底部色條：卡型識別色 + 白K標誌
└─────────────────────────┘
```

### 底部色條規則

- 高度：約佔卡牌總高的 **15%**
- 顏色：對應卡型色（見第 5 節五色系統）
- 內容：卡型名稱 + 卡號 + 白色 K 標誌（左下角）
- **文字顏色：深藍 `#1A2A4A`，禁止用白字**（對比度不足）

### 頂端規則

- **禁止**在頂端放色塊 / color band
- 保留給卡名 + 圖示，讓閱讀空間充足
- 背景：米色 `#FAF7F2`（與整體卡面一致）

---

## 4. 背面（Back）設計規則

- 底色：對應卡型的純色（見第 5 節）
- 中央：白色 K 標誌，置中，大小約佔背面寬度 50%
- **K 綠底（角色卡）特別處理**：白 K 透明度壓至 60-70%，做浮水印感（因為兩者對比度偏低）
- 其他四色：白 K 100% 不透明

---

## 5. 五色系統（正背面對應）

| 卡型 | 正面底條色 | 背面底色 | 色碼 | 視覺感 |
|------|---------|---------|------|------|
| 🎭 角色卡 | K 品牌綠 | K 品牌綠 | `#C8D400` | 最核心，品牌主色 |
| 🎯 任務卡 | 品牌鋼藍 | 品牌鋼藍 | `#7B9CC4` | 指揮感，第二品牌色 |
| ⚡ 技能 / MCP 卡 | 琥珀橙 | 琥珀橙 | `#D4831A` | 工具感、行動感 |
| 📚 知識卡 | 赭紅 | 赭紅 | `#B54030` | 資料庫感、厚重感 |
| 🚨 鐵律 / Power 卡 | 深海藍 | 深海藍 | `#1A2E4A` | 最高權重、壓場 |

**設計原則**：正面底條色 = 背面底色（前後一致，雙面可識別卡型）

---

## 6. 色彩對比規範

| 品牌色 | 疊白字對比度 | 疊深字對比度 | 結論 |
|--------|------------|------------|------|
| K 綠 `#C8D400` | 1.8:1 ❌ | 10.2:1 ✅ | **底條文字必須用深色** |
| 鋼藍 `#7B9CC4` | 2.5:1 ❌ | 6.8:1 ✅ | **底條文字必須用深色** |
| 琥珀橙 `#D4831A` | 2.1:1 ❌ | 8.5:1 ✅ | **底條文字必須用深色** |
| 赭紅 `#B54030` | 3.2:1 ❌ | 6.1:1 ✅ | **底條文字必須用深色** |
| 深海藍 `#1A2E4A` | 11.2:1 ✅ | — | **此色可用白字** |

**底條標準文字色：`#1A2A4A`（深海藍）— 全五色通用**

## 7. K 標誌使用規範

| 資產 | 路徑 | 用途 |
|------|------|------|
| 白色 K，透明底 PNG | `~/Downloads/k-mark-white-transparent.png` | 卡牌背面中央、正面底條左下角 |
| K 綠色，透明底 PNG | `shared/assets/brand/kv-k-mark.png` | 深色底以外場合 |
| K 方塊（綠底白K）| `shared/assets/brand/kv-k-square.png` | 不用於卡牌（太複雜）|

**禁止**：AI 重新繪製或變形 K 標誌

## 8. 字型層級建議

| 層級 | 用途 | 建議大小（76mm 寬卡） |
|------|------|---------------------|
| H1 | 卡名 | 16-18pt Bold |
| H2 | 欄位標題 | 10-11pt SemiBold |
| Body | 欄位內容 | 9-10pt Regular |
| Caption | 底條標籤 / 卡號 | 8pt Medium |

字型優先順序：Noto Sans TC → PingFang TC → 系統黑體

## 9. 印刷規格建議

| 項目 | 規格 |
|------|------|
| 尺寸（含出血）| 82×112mm（各邊加 3mm 出血）|
| 解析度 | 300 DPI 以上 |
| 色彩模式 | CMYK（K 綠色碼 CMYK 需重新確認，RGB `#C8D400` ≈ C18 M0 Y100 K5）|
| 底條色塊 | 建議特別色 Pantone 確認（若批量印刷）|
| 材質 | 300gsm 銅板 + 霧膜（雙面）|
| 圓角 | R4mm |
